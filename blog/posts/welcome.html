Welcome to My Digital Garden

Why I decided to build a portfolio from scratch using vanilla web technologies.

In an era of heavy frameworks and complex build tools, I made a conscious decision to go back to basics. This website isn't built with Next.js, React, or Vue. It's built with Vanilla JavaScript, semantic HTML, and modern CSS.

Why "Digital Garden"?

A portfolio shouldn't just be a static brochures; it should be a living, breathing space where ideas grow. By keeping the tech stack simple, I reduce the friction to write and deploy. No complex hydration errors, no massive bundles—just content and code.

The Stack

  • Vite: For a lightning-fast dev experience.
  • Playwright: To ensure nothing breaks when I push code.
  • GitHub Actions: For CI/CD that just works.
